## 技术细节

### 使用的工具
1. **快速翻译脚本** (`tools/quick_translate.py`)
   - 分段翻译主要 sections
   - 超时时间: 180 秒
   - 成功率: ~80%

2. **模块翻译脚本** (`tools/translate_modules_lang.py`)
   - 逐个翻译 modules
   - 每个 module 独立处理
   - 失败时使用英文 fallback

### API 配置
- **API**: https://api.tiu.me/v1
- **模型**: gemini-2.0-flash
- **Temperature**: 0.3
- **Max Tokens**: 16384-32768
- **超时**: 180-300 秒

### 遇到的问题

#### 1. API 超时
- **原因**: modules 部分内容过大（16 个子模块）
- **解决**: 分模块逐个翻译，每个模块间隔 2 秒

#### 2. nav 部分翻译失败
- **原因**: nav 是空对象 `{}`
- **影响**: 无影响，因为本身就是空的

#### 3. 部分 modules 超时
- **影响**: 使用英文 fallback，不影响网站功能
- **后续**: 可以手动重试失败的模块
