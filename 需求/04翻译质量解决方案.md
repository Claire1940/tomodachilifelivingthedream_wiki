# 翻译质量不稳定问题 - 完整解决方案

## 问题总结

你遇到的翻译质量不稳定问题有四个主要表现：

1. **字段内容错位** - 每个字段的值都是下一个字段的内容
2. **FAQ 问题和答案互换** - question 和 answer 字段内容颠倒
3. **表格数据混乱** - columns 和 rows 的内容混在一起
4. **字段缺失或为空** - 大量字段值为空字符串

## 根本原因

### 值列表翻译模式的固有缺陷

当前的翻译流程：
```
JSON → 提取所有值 → 用换行符连接 → 发送给 AI → AI 翻译 → 按行分割 → 重新组装 JSON
```

**问题所在**：
- AI 模型可能改变行的顺序（特别是表格数据）
- AI 可能合并相似的行或添加空行
- 简单的行数验证和修复逻辑（截断或填充）无法处理这些情况
- 当输出行数与输入不匹配时，重组时会导致错位

### 为什么质量不稳定

1. **API 调用的随机性** - 即使参数相同，不同调用可能产生不同结果
2. **模型的不确定性** - 温度参数、模型版本等都会影响输出
3. **大数据集的复杂性** - 372 个值的翻译容易出错
4. **缺少验证机制** - 没有自动检测和修复的系统

### 方案 4：配置优化建议

#### 4.1 降低 API 随机性

编辑 `transpage_config.json`:
```json
{
  "temperature": 0.3,  // 降低到 0.3（原来可能是 0.7）
  "max_tokens": 8192,
  "top_p": 0.9,        // 添加 top_p 参数
  "frequency_penalty": 0.1,  // 添加频率惩罚
  "presence_penalty": 0.1    // 添加存在惩罚
}
```

**说明**:
- `temperature`: 越低越稳定（0.1-0.3 最稳定）
- `top_p`: 控制采样多样性（0.9 比较稳定）
- `frequency_penalty`: 减少重复
- `presence_penalty`: 鼓励多样性

#### 4.2 增加重试次数

```json
{
  "retry_attempts": 5,  // 从 3 增加到 5
  "retry_delay": 10     // 从 5 增加到 10 秒
}
```

#### 4.3 使用完整 JSON 翻译模式

```json
{
  "use_value_extraction": false  // 已设置为 false
}
```

**优点**:
- AI 可以看到完整的 JSON 结构
- 不会出现值错位问题
- 翻译质量更稳定

**缺点**:
- Token 使用量增加 2-3 倍
- 可能超过 API 限制

### 方案 5：分块翻译策略 (推荐用于大文件)

如果翻译文件很大，可以分块翻译：

```python
# 伪代码
def translate_in_chunks(en_data, target_lang, chunk_size=50):
    """分块翻译大型 JSON"""
    values = extract_values(en_data)

    # 分块
    chunks = [values[i:i+chunk_size] for i in range(0, len(values), chunk_size)]

    # 逐块翻译
    translated_chunks = []
    for chunk in chunks:
        translated = translate_chunk(chunk, target_lang)
        translated_chunks.append(translated)

    # 合并
    all_translated = sum(translated_chunks, [])
    return reconstruct_json(en_data, all_translated)
```

**优点**:
- 减少单次翻译的复杂性
- 降低出错概率
- 可以并行处理

```bash
#!/bin/bash

LANGS="pt,fr,th"
EN_FILE="src/locales/en.json"

echo "Starting translation with validation..."

# 运行翻译
python3 tools/articles/modules/transpage/translate-messages.py \
  --overwrite --lang $LANGS

# 验证每个语言
for lang in ${LANGS//,/ }; do
    echo ""
    echo "Validating $lang..."
    python3 tools/articles/modules/transpage/translation_validator.py \
      $EN_FILE \
      src/locales/$lang.json \
      $lang

    if [ $? -ne 0 ]; then
        echo "ERROR: Validation failed for $lang"
        exit 1
    fi
done

echo ""
echo "All translations validated successfully!"
```
